# Terraform AWS EC2 + Nginx

Prosty projekt Terraform, który uruchamia instancję EC2 na AWS i instaluje Nginx.

## Wymagania

- [Terraform](https://www.terraform.io/downloads.html) ≥ 1.0
- Konto AWS z uprawnieniami do tworzenia EC2
- Klucz SSH do połączenia z instancją

1. Sklonuj repo:
   ```bash
   git clone https://github.com/wudjitzu24/terraform-aws-ec2-nginx.git
   cd terraform-aws-ec2-nginx
